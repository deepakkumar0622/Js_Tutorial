querySelector:
    Returns the element using all like class,id,tagName etcc.

    Example:
            //  queryselector
    var divs=document.querySelector("h1") //tag
    var divs=document.querySelector(".h1") //className
    var divs=document.querySelector("#h1") //Id
    
querySelectorAll:
    Returns a array which respect to the selected class or tag or name
